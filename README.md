IT4040-AI
=========

A project from class IT4040 in HUST

Requirement
-----------

- NodeJS v6.9.0 or later
- NPM v3.10.8 or later

Install Package Dependencies
----------------------------

```bash
npm i
```

Build Project
-------------

```bash
npm run build
```

Run Demo
--------

```
npm start
```

or

```
npm run server
```
