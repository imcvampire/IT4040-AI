IT4040-AI
=========

([English below](#requirement))

Yêu cầu
-------

- NodeJS v6.9.0 or later
- NPM v3.10.8 or later

Cài đặt các gói phần mềm cần thiết
----------------------------------

```bash
npm i
```

Biên dịch chương trình
----------------------

```bash
npm run build
```

Chạy chương trình
-----------------

```
npm start
```

hoặc

```
npm run server
```
